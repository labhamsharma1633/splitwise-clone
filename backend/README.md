# Splitwise Clone

A backend implementation of Splitwise built using Node.js, Express.js, PostgreSQL, Prisma ORM, and JWT Authentication.

## Features

- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Group Creation
- Automatic Group Membership
- PostgreSQL Database
- Prisma ORM

## Tech Stack

- Node.js
- Express.js
- PostgreSQL (Neon)
- Prisma ORM
- JWT
- bcryptjs

## API Endpoints

### Auth

- POST /api/auth/register
- POST /api/auth/login

### Groups

- POST /api/groups
